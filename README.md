# AI Dashboard

## Project Overview

This project is an interactive AI Dashboard built using Streamlit and the Titanic dataset.

The dashboard provides:

* Dataset Overview
* Data Cleaning
* Key Performance Indicators (KPIs)
* Interactive Filters
* Visualizations

## Dataset

Titanic Dataset

Dataset File:
titanic.csv

## Technologies Used

* Python
* Pandas
* Plotly
* Streamlit

## Features

* Dataset Overview
* Data Cleaning
* KPI Cards
* Gender Filter
* Survival Count Visualization
* Passenger Class Distribution
* Age Distribution
* Gender Distribution
* Fare Distribution

## How to Run

1. Install required libraries:

pip install streamlit pandas plotly

2. Open terminal in project folder.

3. Run:

python -m streamlit run app.py

## Output

The application displays:

* Dataset Overview
* Data Cleaning Summary
* KPIs
* Interactive Filters
* Multiple Visualizations
